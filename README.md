# VK Currency Bot

Бот для конвертации валют во ВКонтакте.

## Поддерживаемые валюты
- евро
- доллар
- рубль

## Команды
- /start
- /help
- /values

## Пример использования
```text
евро доллар 100
```
## Настройка токена

Создайте файл `.env` в корне проекта и добавьте:

VK_TOKEN=ваш_токен
